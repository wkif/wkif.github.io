kif的设计美学

设计师导航

http://www.bossdesign.cn/designnav/

设计神器

dopelycolors | 超级在线调色板神器

https://colors.dopely.top/

dopelycolors 超级调色板工具，帮助你探索意向不到的颜色，更快速的将颜色应用到你的设计工作，甚至任何跟颜色有关的东西，专为创意人员提供专业的颜色搭配。你可以创建属于自己的调色板或者探索一个有创意的调色方案，在这里可以管理你的调色板上创建的私人色彩搭配方案，还有很多色彩小工具，比如彩色转换器、对比度检查、颜色梯度生成器等等。

Tabbied | 轻松创建漂亮的艺术涂鸦作品

https://tabbied.com/

Tabbied 通过在线涂鸦设计创造属于你自己的艺术作品，提供 9 款预设的艺术模板，选择任意一个模板开启艺术创作之旅吧。您可以用涂鸦元素创作很多类型的设计作品，比如海报、服饰、卡片、网页设计等等，通过简单的几何图形和颜色激发你的创造力，让您能在一分钟之内就可以创建艺术风格的设计作品。

Adobe 官方出品免费在线抠图神器

https://photoshop.adobe.com/

Freepiker | 国外免费图形资源网站

https://freepiker.com/

Freepiker 由一个小型有创造力的设计团队，通过对设计的热情为动力，提供免费专业的图形资源，致力于每一个项目我们只是热爱创造性的设计，有能力的请支持这样的一个设计网站吧。

Pond5 | 全球最大的免版税多媒体资源网站

https://www.pond5.com/free

Pond5 是全球最大的多媒体资源交易平台。提供海量免费可商用的照片、插画、音乐、音效、AE 模板和 3D 模式、视频等资源。网站上收藏了大量的复古素材，所有的图都是高质量的。我们可以从中搜索到自己需要的插图、照片素材，作为自己的参考。这个网站的一大特点就是有很多复古风格的照片，如果你想在设计中做一些蒸汽波或朋克风格的设计，可以用这些照片作为插图使用。

Wavelry | 绘制双波浪背景图案生成工具

https://wavelry.vercel.app/

Wavelry 一款在线制作双波浪背景图工具，对比 Get Waves 波浪绘制工具多了许多自定义的功能，除了可以设置波浪类型、位置、颜色、方向等等，支持梯度颜色调整、背景颜色调整、以及线性和径向的颜色绘制，如果不喜欢这个波浪类型，按键盘空格键就可以随机生成双波浪图，生成后可以保存为 SVG、PNG 格式，单击即可复制 SVG 代码直接应用到网站上。

Voxelmade | 让人着迷的3D立体像素建模神器

https://www.voxelmade.com/

Voxelmade 由法国导演 Thibault Simar 一手创建的 3d 像素建模软件，喜欢像素风格的朋友一定知道像素创作的乐趣，这里提供了创建 3d 立体像素建模的知识、教程的社区，随时可以根据自己的想象力塑造非常酷的艺术作品，可能是一款让您着魔的 3d 建模神器。

Box Shadows | 在线创建可视化CSS阴影效果神器

https://brumm.af/shadows

Box Shadows 通过手动调整 CSS 盒子的阴影就可生成真实质感的光影效果，我们知道很多时候调整阴影参数数值，很难做到非常自然的光影效果，如果你是网站开发人员，想要创建一个真正的阴影盒子，可能需要花费大量的时间测试，还不一定能够把光影效果细节之处表现出来，因为让一个物体产生阴影，需要考虑轮廓模糊、偏移、颜色、扩展等等，那么 Box Shadows 可视化操作界面，简单调节阴影图层、透明度、渐变反向阿尔法、模糊等等，简单来说，就是不懂 CSS 样式的设计也能创建真实自然的光滑的阴影了，制作完成后直接复制 CSS 代码粘贴到 Style 里面。

Template | 10万+海量精美模板的在线编辑神器

https://www.template.net/

Template 海量设计模板资源站，提供超过 10 万+设计精美，可在线编辑的模板，帮助您更加快速、智能完成工作人文，如果你没有专业设计技能，有了 Template 就可以在线编辑精美模板，整个过程非常简单，只需要寻找合适的模板，在线编辑模板，最后保存或下载设计作品。提供的资源非常丰富，有 Word、Excel、PS、AI、PDF 等等文件格式，可实现多设备即时下载，里面的素材图片、字体、图形都是免版权，全部都由专业设计师创造，非常不错的高级设计模板资源站哦。

Multiavatar | 随机生成12亿个性化头像的神器

https://multiavatar.com/

Multiavatar 一个多元文化的头像在线生成器，同时也是一项免费开源的项目。收录自多个种族、多文化、多个年龄组、多个世界观和各行各业的人。总共可以生成 12 亿加密的有趣头像，这意味着世界上每个人都可以拥有自己独一无二的个性头像，为了说明这个愿景，你可以生成和下载自己的多点世界，帮助您创建自己有趣的社交头像、以及应用程序个性头像，适合全世界每一位有需要的朋友。

CodePen | 全球180万前端设计师的灵感社区

https://codepen.io/

CodePen 顾名思义就是代码笔，是前端设计师测试在线测试代码的神器，同时也是前段设计学习和调试、寻找灵感的网站，全球超过 180 万以上前端设计师在这里分享作品和学习前端技术，除了支持自己的在线调试，还可以查看一些流行炫酷的 js 效果，就是因为 CodePen 大多数的分享的作品是开源，你可以学习到优秀的开发人员是如何构建动画和测试功能，成了世界上最活跃的前端社区，还有很多功能等您挖掘哦。

Unscreen | 智能AI去除视频背景在线神器

https://www.unscreen.com/

Unscreen 智能删除视频背景的神器，可以删除视频或者 GIF 动态图片的背景，支持的格式有[.mp4 .webm .ogg .mov .gif]，可以直接上次您的视频或者 GIF 图片，还支持通过 URL 视频一键 AI 智能轻松帮您的视频去掉背景，再也不用绿色背景来录制视频了，任何的地方都可以录制视频，轻松通过 Unscreen 去掉背景生成高质量的视频，效果非常的惊人，尽管是发丝也能精细的扣出，提供多种背景效果，一键切换后喜欢即可单击下载。

Gradienta | 免费CSS渐变色背景生成工具

https://gradienta.io/

每每看到国外优秀的网站时候，无不被视觉冲击的渐变色吸引，渐变色是给人一种非常有趣的视觉体验，对于网页设计师而言，简单有趣的渐变色 CSS 代码可以给网站增添许多的视觉效果，呈现给用户的体验是非常不错的，如果你正找寻这样的工具，不妨体验下。
