数据结构习题：汉罗塔问题

数据结构习题：汉罗塔问题

六月 14, 2020
  • 习题
  • 数据结构

描述
约19世纪末，在欧州的商店中出售一种智力玩具，在一块铜板上有三根杆，最左边的杆上自上而下、由小到大顺序串着由64个圆盘构成的塔。目的是将最左边杆上的盘全部移到中间的杆上，条件是一次只能移动一个盘，且不允许大盘放在小盘的上面。
这是一个著名的问题，几乎所有的教材上都有这个问题。由于条件是一次只能移动一个盘，且不允许大盘放在小盘上面，所以64个盘的移动次数是：18,446,744,073,709,551,615
这是一个天文数字，若每一微秒可能计算(并不输出)一次移动，那么也需要几乎一百万年。我们仅能找出问题的解决方法并解决较小N值时的汉诺塔，但很难用计算机解决64层的汉诺塔。
假定圆盘从小到大编号为1, 2, …

格式
输入格式
输入为一个整数(小于20）后面跟三个单字符字符串
。整数为盘子的数目，后三个字符表示三个杆子的编号。
输出格式
输出每一步移动盘子的记录。一次移动一行。每次移动的记录为例如 a->3->b 的形式，即把编号为3的盘子从a杆移至b杆。
样例
输入样例
2 a b c
输出样例
a->1->c
a->2->b
c->1->b
限制
时间限制：10000 ms
内存限制：65536 KB
代码

#include <stdio.h>
//将n个圆盘从x柱子上借助y柱子移动到z柱子上
void move(int n, char x, char y, char z)
{
    //printf("B: %d %c %c %c \n", n, x, y, z);
    if (n == 1)
    {
        printf("%c->%d->%c\n", x, n, z);
    }
    else
    {
        move(n - 1, x, z, y); //将n-1个圆盘从x柱子上借助z柱子移动到y柱子上
        printf("%c->%d->%c\n", x, n, z);//将第n个盘子从x移到z上
        move(n - 1, y, x, z);//将n-1个圆盘从y柱子上借助x柱子移动到z柱子上
    }

}


int main()
{
    //分别代表三个柱子
    char ch1,ch2,ch3;
    int n;   //n代表圆盘的个数
    scanf("%d %c %c %c", &n ,& ch1, &ch2, &ch3);
  //  printf("A: %d %c %c %c \n", n, ch1, ch2, ch3);
    move(n, ch1, ch3, ch2);
    return 0;

}
