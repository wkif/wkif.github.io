数据结构习题：求f（x,n）

六月 14, 2020
  • 习题
  • 数据结构

描述

格式
输入格式
第一数是x的值，第二个数是n的值。
输出格式
函数值，保留两位小数。
样例
输入样例
1 2
输出样例
0.40
限制
时间限制：1000 ms
内存限制：65536 KB
代码

#include<stdio.h>
float f(float a, float b);
int main()
{
    double x, n;
    scanf("%lf%lf", &x, &n);
    float e;
    e = f(x, n);
    printf("%.2f\n", e);
    return 0;
}
float f(float a, float b)
{
    float k;
    if (b == 1)
    {
        k = a * 1.00 / (a + 1) * 1.00;
        return k;
    }
    else
    {
        k = a * 1.00 / (b + f(a, b - 1)) * 1.00;

    }   return k;
}
